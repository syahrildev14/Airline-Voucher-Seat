# Airline Voucher Seat

This project is built using Laravel for the backend and Node.js for frontend asset management.

---

## 1. Prerequisites

Make sure you have the following installed:

- PHP 8.2+
- Composer
- React.js
- Node.js (v18+ recommended)
- NPM or Yarn
- SQLite
- Git

## 2. Steps to install dependencies

### backend
```bash
cd backend
composer install
```
### frontend
```bash
cd frontend
npm install
```

## 3. Steps to configure environment

```bash
.env
```

```bash
php artisan key:generate
```

## 4. Steps to run migrations

```bash
php artisan migrate
```

## 5. Steps to run the backend

### backend
```bash
php artisan serve
```
### frontend
```bash
npm run dev
```
